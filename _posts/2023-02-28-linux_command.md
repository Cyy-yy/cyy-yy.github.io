## 1. 查看系统版本

- **查看内核**

    ```bash
    cat /proc/version
    ```

    ```bash
    uname -a
    ```
    
- **查看发行版**

    ```bash
    lsb_release -a
    ```

    ```bash
    cat /etc/issue
    ```

    ```bash
    cat /etc/redhat-release  # 查看Redhat系列linux版本
    ```
